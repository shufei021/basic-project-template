# basic-project-template

> 项目优化模板

## 前端工程自动化

 + API相关 优化
 + 组件相关 优化
 + 指令相关 优化
 + 工具相关 优化

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
